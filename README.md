# Sistema de Buses Inteligentes

Proyecto universitario para la asignatura Desarrollo Backend.

## Descripción
Sistema basado en microservicios. El microservicio de seguridad (MS-Security) gestiona la autenticación y autorización del sistema.

## Integrantes
- Christian Ramirez
- Juan Carlos Alegria
- Ivan Tobar

## Arquitectura
- Frontend: Angular
- Microservicio seguridad: Spring Boot
- Microservicio negocio: Adonis
- Microservicio notificaciones: Flask

## Historias de Usuario implementadas en MS-Security
- HU-007: Registro con email y contraseña
- HU-008: Inicio de sesión con email y contraseña
- HU-009: Control de sesión y prevención de inyecciones URL
- HU-013: Recuperación de contraseña
- HU-004: Autenticación con Google OAuth 2.0

## Entregas
- Entrega 1: módulo de seguridad
- Entrega 2: módulo de negocio
- Entrega 3: monitoreo y comunicación
